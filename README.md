# Money-Gaming
Project description: This project is to automate https://moneygaming.qa.gameaccount.com/ using java to Register a user process and validating the message text that appear under  birthday field if values not entered .

Browsers exe's can be found in Drivers folder in the project. Project can be run chrome and Firefox, can be configured in projectconfig.properties file which can be found src\main\resources.

Instructions to run and build code:

Clone git repository (URL:https://github.com/skumar0326/Money-Gaming.git) using git clone Open project with IntelliJ IDE or Eclipse and download the dependencies. I have developed using IntelliJ using Maven. To run the project you can use either of below options: • Run it through IntelliJ terminal using command--> mvn clean install • Run through command prompt by using command--> mvn clean install • Run it through RunnerTest utility • Run it through the command prompt by going to the project directory • Run it through Intellij->Maven Project->Lifecycle->install.

After the test is finished reports can be found • Extent reports in report folder. • Cucumber json report also found in the project directory

Sample Reports can be found in the Reports Folder

Note: All dependencies are included in project
